# 快速启动指南 - 纯前端 Mock 模式

> 不需要启动后端和数据库，只用 IDEA 打开项目，就能跑起前端看效果。
> 适用于 Windows 系统、0 基础、已安装 IntelliJ IDEA 的同学。

---

## 前提条件

- Windows 10/11
- 已安装 IntelliJ IDEA（Community 或 Ultimate 均可）
- 已经把项目代码下载到本地

---

## 第一步：用 IDEA 打开项目

1. 打开 IDEA
2. 点击 `File` → `Open`
3. 选择项目根目录 `web-develop-hw` 文件夹，点击 `OK`
4. 等待 IDEA 加载完成

---

## 第二步：启动前端

有两种方式，任选其一：

### 方式 A：用 IDEA 内置浏览器（最简单）

1. 在 IDEA 左侧项目目录树中，找到 `frontend/pages/login.html`
2. 右键点击 `login.html`
3. 选择 `Open In` → `Browser` → `Chrome`（或你喜欢的浏览器）

IDEA 会自动启动一个本地服务器并打开页面，直接就能用。

### 方式 B：用 IDEA 终端手动启动

1. 在 IDEA 底部点击 `Terminal` 标签页，打开终端
2. 进入前端目录：
   ```cmd
   cd frontend
   ```
3. 启动 HTTP 服务器（需要电脑上有 Python）：
   ```cmd
   python -m http.server 8081
   ```
4. 打开浏览器访问：http://localhost:8081/pages/login.html

> 如果没装 Python 也没关系，用方式 A 即可。

## 第四步：体验各功能

登录后可以看到导航菜单，点击进入各模块：

| 功能 | 能做什么 |
|------|---------|
| 员工管理 | 添加、编辑、删除员工，搜索、翻页 |
| 部门管理 | 添加、编辑部门 |
| 考勤管理 | 录入、编辑、删除考勤，按员工/月份筛选，查看统计 |

所有数据都是 Mock 的，存在浏览器内存里，**刷新页面数据不会丢失**（登录状态保持），但**关闭浏览器后数据会重置**回初始状态。

## 常见问题

### Q：页面打不开 / 白屏？

用方式 A 右键打开 `login.html`，不要直接双击文件打开。

### Q：登录后跳回登录页？

清除浏览器缓存（`Ctrl + Shift + Delete`），然后重新登录。

### Q：数据没了？

Mock 数据存在内存里，关闭浏览器标签页后会重置。重新登录即可看到初始数据。

### Q：想恢复初始数据？

关闭所有该网站的浏览器标签页，重新打开 `login.html` 登录即可。

## 架构说明

当前是 **纯前端 Mock 模式**，不依赖任何后端服务：

```
┌──────────────────────────────────┐
│          Chrome 浏览器            │
│   http://localhost:xxxx/pages/   │
├──────────────────────────────────┤
│     前端 HTML + JS + CSS         │
│                                  │
│   api.js 中的 Mock 数据           │
│   （替代了真实的后端 API）         │
│   所有增删改查都在内存中完成       │
└──────────────────────────────────┘

不需要：× 后端 Spring Boot
不需要：× 数据库 MySQL
不需要：× Docker
不需要：× Java / Maven
```

如果后续需要连接真实后端，把 `frontend/js/api.js` 替换回原版即可。
